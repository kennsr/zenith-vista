# The Zenith - Luxury Manhattan Living

A premium real estate website showcasing luxury penthouse residences in Manhattan.

## About

The Zenith represents the pinnacle of Manhattan living, featuring world-class amenities and breathtaking views. This is a demo/portfolio website.

## Tech Stack

- **React** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **shadcn/ui** for UI components

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Deployment

The site is configured to deploy at the `/zenith-vista` base path.
